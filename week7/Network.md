#### HTTP
- The protocol that allows resources to be requested and sent on the internet 
- Types of requests
    - GET
    - POST
    - PUT & PATCH
    - DELETE
- Types of Responses 
    - 100-199: Informational Responses
    - 200-299: Successful Responses
        - 200 - OK
        - 201 - Created
